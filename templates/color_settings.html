🎨 色設定

各メンバーのカレンダー表示色を確認できます。
色を変更するには app.py の FAMILY_MEMBERS を編集してください。

{% for name, info in family_members.items() %}
{{ name }}
文字色： {{ info.color }}
背景色： {{ info.bg }}
{% endfor %}

📝 色の変更方法

FAMILY_MEMBERS = {
    'りゅうや': {'color': '#FF6B6B', 'bg': '#FFE5E5'},
    'りょうや': {'color': '#4ECDC4', 'bg': '#E5F9F7'},
    ...
}

color = 文字色・アクセント色
bg = イベントの背景色
